This directory contains ERA5 reanalysis wind data (.nc format).

Place your files here:
- era5_wind_T0.nc    – ERA5 wind at observation time T0

Variables expected:
- u    – Zonal wind component (m/s)
- v    – Meridional wind component (m/s)

Download ERA5 data from: https://cds.climate.copernicus.eu/
(requires free CDS API account)

If no ERA5 data is available, SAT-GUARDIAN automatically generates
synthetic wind fields for demo purposes.
