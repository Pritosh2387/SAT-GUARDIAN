# SAT-GUARDIAN 🛰️

Physics-Aware Adaptive Satellite Frame Interpolation System

Transforming 30-minute INSAT-3DS observations into adaptive 7.5-minute weather intelligence using physics-constrained AI.

## Features

- Physics-constrained optical flow using ERA5 wind fields
- Cascaded three-frame generation (T0.25, T0.50, T0.75)
- Pixel-wise confidence maps
- Cloud Motion Consistency Score
- NetCDF (.nc) support
- Interactive visualizations and GIF generation
- End-to-end inference pipeline
